# Project Babel Image Guideline

본 가이드는 [Project: Babel]의 일관된 시각적 경험을 위해 아이템, 스티커, 배경 및 캐릭터 이미지의 특징과 생성 프롬프트 규칙을 정의합니다.

## 🎨 전체 시각적 스타일 (Overall Style)

*   **아트 스타일**: 귀엽고 부드러운 2D 애니메이션/일러스트 스타일 (Soft 2D Anime/Illustration).
*   **색상 팔레트**: 파스텔 톤의 보라색, 핑크색, 하늘색이 주를 이루며, 무지개빛 광택(Iridescent/Holographic sheen)이 특징입니다.
*   **조명**: 부드러운 글로우(Glow) 효과와 네온 라인이 포함되어 미래적이지만 따뜻한 느낌을 줍니다.
*   **디테일**: 반짝이는 별, 하트 아이콘, 작은 기하학적 파티클이 공중에 떠다니는 장식적인 요소가 자주 사용됩니다.

## 🎁 아이템 이미지 가이드 (Item Image)

아이템은 게임 내 인벤토리와 대화창에서 보상으로 노출되는 독립적인 오브젝트입니다.

*   **구성**: 흰색 배경(White background)에 단일 오브젝트가 중앙에 배치됩니다.
*   **형태**: 약간의 부피감이 느껴지는 2D 일러스트이며, 오브젝트 주변에 반짝이는 효과(Sparkles)와 파스텔 톤 파티클이 배치됩니다.
*   **재질**: 금속이나 플라스틱 재질이라도 부드러운 곡선 위주로 디자인하며, 표면에 무지개빛 반사(Iridescent sheen)를 추가합니다.
*   **프롬프트 핵심 키워드**:
    *   `Soft 2D illustration`, `cute stylized game icon`, `iridescent sheen`, `pastel colors`, `sparkles and stars particles`, `white background`, `clean thick outlines`, `high quality`.

## 🏷️ 스티커 이미지 가이드 (Sticker Image)

스티커는 스티커북에 수집되는 보상으로, 아이템보다 조금 더 평면적이고 '수집품' 같은 느낌을 줍니다.

*   **구성**: 원형 또는 오브젝트의 외곽선을 따라 굵은 흰색 테두리(Thick white border)가 있는 스티커 형태입니다.
*   **스타일**: 아이템보다 조금 더 단순화된 플랫 벡터(Flat vector) 느낌을 주되, 전체적인 파스텔 톤과 반짝이 효과는 유지합니다.
*   **프롬프트 핵심 키워드**:
    *   `Circular sticker design`, `die-cut sticker with thick white border`, `flat vector illustration`, `cute mascot style`, `pastel palette`, `game reward sticker`, `isolated on white`.

## 🖼️ 배경 및 캐릭터 가이드 (Reference Only)

*   **배경**: 곡선 위주의 인테리어, 홀로그램 디스플레이, 파스텔 톤 네온 조명이 포함된 SF 공간.
*   **캐릭터**: 동글동글한 로봇 디자인, 표정이 풍부한 스크린 얼굴, 금속 표면의 무지개빛 광택.
