# Stage 4: 신호등 게임 (Open and Close)

[상황 설명]
커다란 철문 앞에 도착했어요. 이 문은 특이하게도 "열려라!"와 "닫혀라!"라는 말을 알아들어야 열린대요.

[Main Dialogue (대화)]
Buddy: "This door is a bit funny. It only opens when you say the right word."
(이 문은 좀 웃겨. 네가 알맞은 말을 해야만 열리거든.)

You: "What should I say?"
(내가 뭐라고 해야 해?)

Buddy: "The door is closed now. Say 'Open!'"
(지금 문이 닫혀 있어. '열려라!'라고 해봐.)

You: "Open! Look, it is moving!"
(열려라! 봐, 문이 움직여!)

Buddy: "Great! Now, if a bad robot comes, say 'Close!' very fast."
(잘했어! 이제 나쁜 로봇이 오면, 아주 빨리 '닫혀라!'라고 말해.)
