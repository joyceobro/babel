# Stage 75: 버디와 토론하기 (I Think So)

[상황 설명]
어떤 길로 가는 것이 더 빠를지 버디와 의견을 나누고 있어요. 내 생각을 말해볼까요?

[Main Dialogue (대화)]
Buddy: "**I think** the left path is better. What do you think?"
(내 생각엔 왼쪽 길이 더 나은 것 같아. 넌 어떻게 생각하니?)

You: "**In my opinion**, the right path is faster."
( **내 의견으로는**, 오른쪽 길이 더 빨라.)

Buddy: "**I think** you are right. Look at the map again."
(네 말이 맞는 것 같아. 지도를 다시 봐봐.)

You: "Yes! **I think** we should go right."
(응! 내 생각에 우린 오른쪽으로 가야 해.)

Buddy: "**I think** so, too. Let's go!"
(나도 그렇게 생각해. 가자!)
