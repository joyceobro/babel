# Stage 24: 강아지 산책 (Walking the Robot Dog)

[상황 설명]
마을 공원에서 로봇 강아지를 산책시키고 있는 친구를 만났어요. 여러 마리의 로봇 동물들이 있네요!

[Main Dialogue (대화)]
Buddy: "Look at the **dog**! It is a robot **dog**."
(저 **개** 좀 봐! 로봇 **개**야.)

You: "It is so cute! Woof, woof! Is that a **cat**, too?"
(정말 귀엽다! 멍멍! 저건 **고양이**니?)

Buddy: "Yes, that is a robot **cat**. It says meow!"
(응, 저건 로봇 **고양이**야. 야옹 하고 울지!)

You: "I like the robot **bird**. It is flying!"
(난 로봇 **새**가 좋아. 날고 있잖아!)

Buddy: "Wow, there is even a robot **rabbit**! It can jump."
(와, 로봇 **토끼**도 있어! 점프를 할 수 있네.)
